# 方案书 · 调试场优化（测试人员反馈不方便）

**日期** 2026-07-24 ·**状态** 待拍板 ·**来源** 用户转测试人员反馈 +「你自己走一遍流程看看」

## 现状（我截图走了一遍，`C:/tmp/dbg_arena.png`）
左上角"🛠调试场"大面板，占了战场 **~40%**（从顶到中部），`mouse_filter=STOP` 吃点击、**没有关闭/折叠**。内容：选龟/选边、小将、假人HP、装备星级、倍速、开始/编辑/再来、清空/返回。

## 痛点（用户点名·核心是【摆放流程】）
1. **★选龟是模态弹窗，摆放流程极累**：点"选龟"→弹出满屏 28 龟网格(`_edit_open_turtle_grid`→`_edit_make_popup` 全屏遮罩)→点一只→`_edit_pick_turtle` **自动关弹窗**→回去点地摆一只。**每换一种龟都得重开弹窗一次**，弹窗还盖住整个战场看不见场。摆一支杂队 = 开关弹窗几十次。
2. **面板不能关** → 大黄框占左半场 ~40%、吃点击，左边摆不了、也挡视野。
3. **训龟大师完全不能测** → `_spawn_trainers()` L4193 `if DEBUG_EDIT: return` 故意跳过。

## 方案
### ★核心 · 选龟改成【常驻笔刷栏】（治摆放流程）
- 去掉模态弹窗。改成一条**常驻可横滚的笔刷栏**（屏幕底部一条）：28 龟 + **小将 3 种**（近战·浪板 / 远程·火箭 / **精英小将**）+ 🐢训龟大师，每格是头像/图标。
- **★顺带修 bug**：现在 `_edit_place_unit` 写死 `elite: false`，**精英小将根本摆不出来**（用户2026-07-24 发现）。加 `_edit_minion_kind = front/back/elite`，elite → `_make_unit("__minion__", …, {"minion":true,"elite":true,…})`。
- **点一格 = 选为当前笔刷**(高亮框)，直接在场上**连点连摆**、要几只摆几只；换单位就点栏里另一格 —— **中间不开关任何弹窗、不挡战场**。
- 队伍(左/右)也放笔刷栏边上一键切。

### A · 大控制面板可折叠
- 低频设置(假人HP/星级/倍速/满龟能/开始·编辑·再来·清空)收进**可折叠面板**，顶栏加 ⊟/Tab 收起 → 只留左上一个小"🛠▶"。
- 折叠时释放战场(面板 `visible=false`、STOP 范围缩)；折叠态**跨 UI 重建持久**。

### ★核心2 · 单龟配置面板（点已摆的龟 → 逐只配·治"全局/不方便"）
现在全是**全局设置**、单只配不了 —— 用户点名的痛：
- **技能**：调试场**根本不能选技能**（摆下的龟用别处 loadout）→ 补：列该龟 skillPool 候选技，点一个设主动。
- **血量**：现在只有全局假人HP ±100 →改：选中龟直接填/拖血量，各龟独立。
- **无敌**：现在全局"掉血/不死"开关 →改：每只龟一个"无敌"开关。
- **星级**：现在全局 ★1/★2/★3 →改：跟着当前选中龟。
- **装备**：已有(选中→加装备连加)，并入本面板；每件带星级。
- **满龟能**：现在全局"满龟能:关" →改：每只龟一个开关（开=满充能，秒放技看效果）。
- （可选）等级。
点一只龟 → 它旁边/侧边冒出这个小面板；点空地=继续摆下一只。

### B · 训龟大师能测（作为笔刷栏一项·自由摆位）
- 笔刷栏里点 🐢训龟大师 → 点空地摆一只（跳过"蛋后上方"站位逻辑）。
- 加**主动技能选择器**（钩锁/怒火药水/口哨/冰川）设 `_tr_active` 随时切；被动(无/魔法石)。
- 左侧大师按 **Q** 放（已接）；不加清CD按钮（用户定）。

## 已知风险与未决点
- **4193 跳过是故意的**：因为正式对局大师站位挂"蛋的后上方"，调试场没蛋/没双路。→ 方案：调试场大师**自由摆位**、跳过站位逻辑。（未决①见下）
- **大师是场外监视者**（不被索敌、伤害降为1）：调试场里放它，敌方假人不会揍它（对）；要能看到它技能命中假人。
- **CD 14–20 秒**，等着放技能慢。→ 未决②：要不要加"清CD/立即放"按钮。
- **门禁**：加 `verify_debug_arena`（守"调试场能生成大师 + 面板折叠 API 存在"），headless 只能守逻辑层，观感要人看。
- 纯调试工具，**不动任何玩法数值**。
