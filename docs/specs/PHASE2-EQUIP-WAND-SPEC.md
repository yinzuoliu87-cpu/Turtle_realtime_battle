> 钩子名（on_turn_begin / on_side_end / on_cast）与"N 回合"时长都是回合制概念，实时版是秒级 tick。
> 部分装备的名字/属性/效果在后续评审中已改。（例：p2eq_022 本文写"余烬法杖·+20最大龟能·on_cast·真火1回合"，实际是"余烬燃油瓶·充能速率+10%·每8秒抛火瓶·真火5秒"）

## p2eq_022 🔮余烬法杖 — 费1 [元素/杖·火]
- 属性: +8/15/25攻 · +20最大龟能。
- 效果(on_cast 施法后): 对目标施加 (20+10%ATK)/(35+15%ATK)/(60+20%ATK) 层灼烧; 同时挂「真火」状态1回合。
- **真火**: 使其受到的灼烧伤害 → 转真实伤害 (新状态, 需 Dot burn 结算时查 _trueFire flag → dmg_type true)。

## p2eq_023 🔥灼热火珊瑚 — 费3 [元素] (=phase1 e_fire 强化版)
- 属性: +40/60/80生命 · +10/25/40攻 · +5/8/13魔穿。
- 效果(on_hit 逐段): 每段攻击(技能/装备等)额外施加 (5+7%ATK)/(7+11%ATK)/(10+15%ATK) 层灼烧。
- **3★ 主动 (on_turn_begin, 2026-06-24 补)**: 满法力(龟能满 _energy>=_maxEnergy)→ 对全敌各 60 灼烧, 放完清空龟能。
  - ⚠️ 携带者无龟能系统(_maxEnergy==0)时无"法力"可满 → 简化为每 3 回合触发一次 (counter fallback, _p2FireCoralTurnCtr)。这是【有意分歧】: 双路无独立法力条, 借龟能近似"满法力"。

## p2eq_024 🐲龙蛋 — 费5 [元素] (=phase1 e_dragon_egg, 数值升级)
- 属性: +30/55/300攻 · +15/25/50魔穿 · +20最大龟能。
- 吐息(on_turn_begin): 每回合+1层, 达3层→喷火龙沿一排(随机有敌列, 同列F+B)飞过, 重置层数; 装备即3层。
  - 同列友军: 回复 (70+0.7ATK)/(150+1ATK)/(1000+2ATK) 生命 (走 applyHeal/healAmp)。
  - 同列敌军: (50+0.7ATK)/(120+1ATK)/(1500+2ATK) 魔法(走on-hit链) + (30+10%ATK)/(50+15%ATK)/(100+15%ATK) 灼烧层。

## p2eq_029 ❄️冰封水母 — 费3 [元素] (=phase1 e_jelly 强化版)
- 属性: +50/110/150生命 · +9/15/25护甲。
- 效果(on_hit 逐段): 每段攻击(技能/装备) 20/25/30%概率 → 额外 10/15/25 魔法 + 冻结眩晕1回合(已眩晕则不再叠眩晕)。每成功冻结一次 → 携带者 +20/30/50护盾(走 grant_shield)。
