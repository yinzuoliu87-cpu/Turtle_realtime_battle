- 放招 = 龟能充能（非回合制 CD）；龟能/决胜局/装备套装学派 都是改进版
- 灼烧等 DoT = 层数衰减模型；真伤吃暴击；资源条字段镜像对齐 HpBar 端口
- **装备 = 固定值 + 只吃星级 ×1.8，不随赛季等级缩放**（对齐回合制；2026-07-01 曾误做等级缩放已撤）
- 召唤体吃等级 = 仅固定值召唤（随从/海螺虫/大熊）×(1+5%/级)，owner 派生的不重复缩放（2026-07-01 用户确认保留；realtime 独有，回合制无）
- **商店售价 = 装备 cost(费) 1:1 深海币**（2026-07-01 用户定；原 ×3 占位）；卖价独立
- **3合1 合成 = 背包 + 龟身装备一起算**（局外 `auto_merge_all`）；含龟身件→升星件放回原龟（2026-07-01 用户提的边界例）
- on-cast 装备效果在实时 = 龟能满自动高频触发（非回合制手动一次）；永久类（加特林减甲）加每目标累计上限防削到 0
- 局外/战斗数值多为草案值，待 F5 调手感
- **弹道方向 = 逐件审查时顺手修**（2026-07-01 用户定）：很多移动弹体是 billboard（永远正对相机=没方向），不专门铺一轮全量改，而是**每审到一件带移动弹道的装备/技能就把它的弹道方向一起修**。有向弹道做法：躺平贴地 `billboard=DISABLED; axis=AXIS_Y` + `rotation.y = -atan2(dir.y,dir.x)`（±90° 调长轴垂直/平行行进），范例 `_lava_volcano_erupt`。
- **装备/技能审查流程**（2026-07-01 用户定）：一件一件走，实现前先问清机制细节（数值单位/方向/位移量/时机），别假设；每件专属演示配置登记 `docs/design/装备测试配置.md`，调 EQDEMO 环境变量实际跑起来眼验。
