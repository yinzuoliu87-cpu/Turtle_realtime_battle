# 实时版 2.5D 战斗架构方案

> 决策 2026-06-27: 战斗表现层从 2D(假高度) pivot 到 **2.5D**(3D空间 + 2D立绘billboard + 真高度/影子/物理)。根治"击飞/飞起来"靠影子戏法凑合的问题。视觉调性不变(3/4斜俯视·卡通深海, 见 [[视觉方向方案]]),改的是"真3D实现"而非"2D假"。

## 一、核心架构 (Godot 4 3D)
| 部件 | 做法 |
|---|---|
| 场景根 | `Node3D`(新 `scenes/RealtimeBattle3D.tscn` + `RealtimeBattle3DScene.gd`) |
| 相机 | `Camera3D` 固定 **3/4 俯角**(~50° 看下来), 透视(轻微) — Botworld/八方旅人那种 |
| 竞技场 | 地面 = `MeshInstance3D`(PlaneMesh) + 深海贴图; 单位在 **XZ 平面**移动, **Y=高度** |
| 单位 | `Sprite3D`(**billboard 模式**, 立绘贴图) 永远朝相机; 位置(x,y,z), y=0贴地, y>0被击飞抬起 |
| 影子 | **blob 暗影**(地面压扁的暗 Sprite3D/Decal, 跟 XZ 不跟 Y); 单位 y 越高影越小越淡 = **真高度驱动**(非2D假)。billboard游戏标配(真光影打billboard难看) |
| 血条/UI | 2D `CanvasLayer` overlay + `camera.unproject_position(世界坐标)` 定位到屏幕; 或 Label3D billboard |
| VFX | `GPUParticles3D` / billboard 特效 在3D空间 |
| 击飞 | 给 **y 初速 + 重力** → 真抛物弧线, 落回 y=0; 横向同时滑 → 真"飞出去再砸地" |

## 二、逻辑复用 (关键: 最难最值钱的不浪费)
- **效果引擎 / 28技能 / 59装备 / 层数DoT / 索敌 / AI = 照搬**。位置 `Vector2 → XZ 平面`(Vector3 的 x,z); 距离/移动/索敌用 XZ 算, 数学完全一样。
- **局外 UI(菜单/商店/背包/赛季) = 完全不动**(2D)。
- **只重搭**: 单位表现(Sprite3D billboard + blob影 + 血条overlay) · 相机 · 移动应用(XZ + Y物理) · VFX · 击飞物理。

## 三、坐标约定 (让现有逻辑几乎不改)
- 单位仍用 `pos: Vector2` 当 **XZ 平面坐标**(x=x, z=y); 另存 `height: float`(Y) + `vy: float`(竖直速度) 给击飞物理。
- 世界坐标: `node3d.position = Vector3(pos.x * WS, height, pos.y * WS)`(WS=像素→米比例, 如 0.02)。
- → 现有大量 XZ 逻辑(索敌/移动/距离/范围/直线)**几乎零改**, 只新增 Y 维度给高度/击飞。
- ARENA 边界(现 Rect2 像素)→ 同比例映射成 3D 地面范围。

## 四、分阶段 (每阶段验证, 现有2D版保留到3D验OK再切)
1. **骨架验证**: 新3D场景 + Camera3D 3/4 + 地面plane + 2个billboard龟 + blob影 + **击飞抛物 demo** → 截图验"真飞起来"观感对不对。(先不接逻辑,纯验画面)
2. **单位系统**: `_make_unit` 改 Sprite3D billboard + blob影 + 血条overlay; pos→XZ世界坐标映射。
3. **逻辑接入**: 移动(XZ) + 攻击 + 效果引擎(复用) + 索敌/AI(复用) → 完整可打。
4. **击飞物理 + juice**: 真抛物 + 影随高缩放淡 + squash/stretch(billboard scale) + 受击闪白 + 顿帧 + 震屏(摇Camera3D)。
5. **VFX 3D + 竞技场美术 + 打磨 + 音效**。

## 五、风险 / 诚实边界
- **billboard 透明排序**: Sprite3D 透明像素排序要调(alpha_cut/render priority), 防穿插闪烁。
- **血条 overlay 投影**: unproject 每帧更新; 单位被遮挡/出屏要处理。
- **移动端性能**: billboard 很轻, 3D 地面+少量光, 可控; 早测一次。
- **美术**: 地面贴图/VFX/竞技场/光照需素材 → 我做系统+占位标注, 素材待用户/美术。
- **保留现有2D版**(`RealtimeBattle.tscn`)直到3D版验证完整可玩再切主菜单入口, 不马上删。
