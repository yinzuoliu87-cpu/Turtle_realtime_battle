## 一、设计目标（解决现版的病）
| 现版病 | V2 怎么治 |
|---|---|
| 羁绊乱/很多没意义 | 装备/羁绊全挪局外慢慢配 → 从战斗噪音变成**赛前策略** |
| 装备战斗中突然装/生星违和 | 装备全局外，**战斗进场即固定阵容**，无中途装备/合星 |
| PvE 打随机 AI 没魂 | **异步/同步 PvP** 打真人阵容，有对手/排名/赌注 |
| 双路像 double grind | 每场赌**一条命**，有 stake；越输越有翻盘资源 |

## 二、核心循环（三层）

### 赛季层（5 天一轮）
- **开赛重置**：每人 **3 统领 / 0 深海币 / 0 物品 / 8 条命**。
- **进程**：不断打 PvP → **输一场 -1 命** → **0 命 = 本赛季淘汰**。
- **5 天结束**：赛季结算（按击杀龟蛋总数排名（单一榜，详见 §十五#3）+ 奖励），重置开新赛季。

### 对战层（一场 PvP）
- **进场前**：玩家**手动分路**（3 统领分上路/下路）+ 准备。
- **对手 = 另一玩家**：
  - **MVP 全异步（打 ghost 快照）**：对方阵容由 **AI 托管**（自动分路 + 战斗内自动放技能），用对方存档的出战阵容。
  - **同步（在线、双方实时操作）留后续需求再评估**（见 §十三/§十五）。
- **战斗**：延续**双路**（上路 → 下路 → 终极战场），凿穿对方蛋。
- **结果**：输 → 失一颗心。

### 养成层（局外，战斗之间）
- **深海币商店**（局外）：花深海币买 装备/物品。
- **配队**：给统领 装备 / 合星 / 凑羁绊，存一套"出战阵容"。
- 无计时，慢慢想。

## 三、经济（重做）
- **取消局内每回合 +币 与利息**（战斗中不再有商店/经济/合星）。
- **战斗后获得深海币** = `25 + 2×现有心 + 5×已失心 +（赢则 +15）`（与 §十五#2 一致）
  - 满血 8 心 → 25+16+0 = **41**；输到 4 心 → 25+8+20 = **53**；1 心 → 25+2+35 = **62**
  - 效果：**越输越多币**（comeback），给落后者翻盘资源（同云顶连败经济逻辑）。
- 深海币只在**局外商店**花。

## 四、改造清单（现有内容：留 / 挪 / 砍 / 新建）
- **留（复用）**：28 龟 · 140 技能 · 龟能 · 59 装备 · 羁绊（注：synergies.gd 10 pet-tag 双路本不生效→废，见 §十四）· 双路战斗引擎 · 决胜局 · 蛋 · 小将 · 分路。
- **挪到局外**：装备商店 · 买装 · 合星 · 凑羁绊 → 全部从战斗中移到养成层。
- **砍**：局内每回合经济/利息 · 战斗中商店浮层 · 战斗中拖装/合星。
- **新建**：赛季/生命系统 · 异步 PvP（对手阵容存档 + AI 托管）· 局外深海币商店 · 战斗后结算币 · 匹配。

> 关键认知：**大体内容留得住**（龟/技能/战斗/双路引擎），换的是**外壳**(什么时候 build + 对手是谁 + 赌注是命)；但 **META 层（经济/build/学派框架）+ 小将是真改造**，不是纯换皮（见 §十四对账）。

### 匹配 + 异步（公平性设计 · 参考 Super Auto Pets 的 ghost 模型）
**核心：按进度配对，双方资源相当 → 公平。**
- **对手 = 其他玩家阵容快照（ghost）**，从**同进度档**池抽：同"总战斗数档"（= 同装备槽上限）+ 相近 局内等级/心数。
- **全异步（已定 2026-06-26）**：对手**永远是快照 ghost + AI 托管**（分路 + 放技能），**不做实时同步**。
  - 理由：回合制双路同步 = 对方长回合时你干等 + 需服务器/掉线处理；而同步**不改善你的体验**（操作自己阵容方式一样）。同步留作后续需求再评估。
- **快照内容**：对方某次在该档时存的出战阵容（3 龟 + 装备 + 分路）。打的是"录像式队伍"，不需对方在线。
- **为什么公平**：同总战斗数 = 同装备槽 = 同资源上限；连败方靠 comeback 经济补装备 → 不会被碾。
- **待拍参数**：配对只按"总战斗数档"，还是再叠 心数/排名（让强弱更接近）？(SAP 只按 turn/进度、接受一定变数；叠心数更稳但匹配池变小)

## 十、阵容配置细则 + 装备绑定（第六轮已定 + 关键实现注意）

### 已解（第六轮，解掉上面 A1 / C3 / D4 / E5）
- **奖励**：赢/输**都给深海币**(`25+2×心+5×失心`)；**赢保持心数、输 -1 心**。（"物品"= 币，统一了）
- **首战**：第一场就是 3 裸龟+小将按背包预设分路登场 = **有意的热身战**。
- **小将装备 = 跟龟统领一样在背包配**（玩家给小将占位配装备）。→ 解了"小将装备大坑"。

### 阵容配置细则（背包上部，实时）
- 每路固定 **3 占位**。玩家摆 3 龟到两路（如 左 2 龟 + 右 1 龟），**空位自动补小将**（左 1 小将 + 右 2 小将）。
- **移龟换路 = 实时挤位**：把右边的龟移到左边 → 左边的小将被挤到右边(下战场)。两路始终各 3。
- **小将位置可调**：放前排 = 前排小将(砍 1.4×)、放后排 = 后排小将(射 1.5×)。
- **某路 0 龟统领 → 该路一只小将实时变「精英」**(整排均摊)。配置时实时刷新。

### ⚠ 装备绑定（关键 · 用户点名"别吞装备/异常"）
**核心 `[已定]`：绑占位 —— 龟装备跟本体、小将装备绑占位、被挤出退背包池。**
- **龟的装备**：跟**龟本体**走 —— 龟换路，装备跟着到另一路。
- **小将的装备**：小将是**占位**（玩家配"前排小将槽"的装备），装备应跟**占位/位置**走。
- **冲突点**：龟移入一路把小将挤出去时，被挤小将的装备跟谁？精英化(整排均摊)会不会动到装备归属？
- **实现铁律（防吞）**：实时重排时，每件装备明确归属一个持有者（龟实例 or 小将占位），重排只搬"持有者"、装备引用跟着搬，**绝不凭空消失/复制**；打完回背包时**全收**（含被挤出/精英化/死亡的）。
- **已定**：小将被挤出/displaced 时，其装备**退回管理区（背包下部）** —— 不跟小将走、不留占位。这是最干净的防吞：占位没了装备就回背包池，玩家重配前再装上。龟装备仍跟**龟本体**走（换路带着）。
- **防吞逻辑齐了**：① 龟换路 → 装备跟龟；② 小将被挤出 → 装备退背包池；③ 打完 → 全收（含死的/精英化/被挤出的）。任何时刻装备要么在某持有者身上、要么在背包池，绝不悬空消失。

## 十二、消耗品 + 赛季结束/0命 + 调试（第七轮已定）

- **0 命 = 本赛季结束**（该玩家出局）。
- **赛季结束/重置**：所有东西重置 —— **商店 + 背包清空**；商店**锁在结束时的 10 卡状态、全显"已结束"**，直到下一大轮才重新开放。币/心/局内等级/装备 全清。
- **调试功能**：加个"**快进时间到下一大轮前 20 秒**"（dev/调试模式用，方便测赛季切换/重置，不用真等 5 天）。

## 十三、匹配怎么做（异步 ghost 机制）
**核心：打的是"快照(ghost)"，不是实时对局。**
- **存什么**：玩家每次配好出战阵容（3 龟 + 装备 + 分路 + 进度档）后，上传一份**快照(ghost)** 到服务器，按"总战斗数档"归类。
- **配对**：点"在线匹配" → 服务器从**你同进度档**（同总战斗数 = 同槽）的池里抽一个 ghost 给你（随机、排除自己/近期打过的）。
- **打**：打那个 ghost 的队；ghost 的分路是它存的、战斗内技能由 **AI 托管**释放。
- **单向**：结果**只影响你**（你的心/币），被你打的 ghost 主人**不受任何影响**（它只是个"录像队伍"）。
- **公平**：同档 = 同槽 = 同资源上限 + comeback 经济。

### 实现要点
- **需一个轻量后端**：存/取 ghost 快照 + 匹配接口。**非实时联机**（无帧同步/服务器权威战斗），只是"存档案 + 抽档案"，工程量小。（注：带排行榜防作弊要后端复算，见 §十五#3，非超轻）
- **冷启动（池子空）**：上线初期没真人 ghost → 用**预设 bot 队**按各进度档填充；真人多了自动换成真人 ghost。
- **保鲜/防重复**：优先抽近期快照；记录近期打过的、别重复打同一个 ghost。
- **离线可玩**：打的是快照，对方不用在线，玩家随时能匹配。

### 进度档分档（解决"高战斗数池子太小/极端情况"）
战斗数低时按"槽位断点"分**窄档**（保证同槽公平）；战斗数高时（槽已满 4）分**宽档**（保证池子有人）：

| 总战斗数 | 槽 | 匹配档 |
|---|---|---|
| 0-1 | 0 | 档 0 |
| 2-3 | 1 | 档 1 |
| 4-5 | 2 | 档 2 |
| 6-8 | 3 | 档 3 |
| 9-14 | 4 | 档 4 |
| 15-20 | 4 | 档 5 |
| 21-30 | 4 | 档 6 |
| 31-40 | 4 | 档 7 |
| 41+ | 4 | 档 8 |

（9-14 是补的过渡档；15+ 按你说的越分越宽。低档窄 = 同槽公平，高档宽 = 保池子够人。）

### ⚠ 极端情况兜底
- **某档池子空/太薄 → 抽 bot 队**（预设、按该档资源配）。bot **不只是冷启动，是永久安全网** —— 任何档随时凑不到真人 ghost 就用 bot，保证永远匹配得到、不卡住。
- **个别玩家遥遥领先**（别人都 10 场、他 41+）→ 高档没真人 → 打 bot。
- bot 队强度按该档资源（同槽）配，不强不弱，维持公平。
