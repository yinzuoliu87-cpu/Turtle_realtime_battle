## 1. 小龟 (basic) — C级 · 近战
**基础**：HP 945 · ATK 40 · DEF 14 · MR 13 · 射程 70 · 移速 105 · 攻速间隔 0.85s

**被动**
- **不屈**：造成任何伤害按目标稀有度增伤 — C +20% / B +23% / A +26% / S +29% / SS +32% / SSS +34%（覆盖普攻/技能/真伤/固伤，每次只算一次）
- **强化普攻（每6秒）**：每 6 秒强化下一次普攻 → 附 0.7×ATK + 目标已损生命 20% 物理 + 击飞 + 自身获盾

**普攻 · 攻击**：1.0×ATK 物理（近战）

**3选1**
| 技能 | 龟能 | 完整效果 |
|---|---|---|
| **打击** | 80 | 全程定身；10 波，每波 0.15s；每波随机挑 1 存活敌为方向，发气波（慢飞·**可被挡**：命中路径第一个敌/蛋，障碍穿过、我方不挡）；总 4×ATK 物理（每波 0.4A·吃不屈）；10 波放完龟能才开始充能 |
| **龟派气波** | 100 | 蓄力（蓄力时可选一次 ≤300码 位移冲刺，奔向能命中最多敌的位置，不贴脸、考虑碰撞体积）→ 先给自身 +buff(3秒) → 发一道气波缓慢飞到尽头，命中 1 段 3.5×ATK 物理 + 击飞 + 击退（较远较久）；须先自增 buff，该气波才吃到增伤 |
| **过肩摔** | 120 | Sett R 式：抓主目标 → 挑空 → 摔到小龟背后 ~60码 + 眩晕 0.5s；主目标受 0.7×ATK + 26%目标最大生命；周围 250码其余敌受 0.2×ATK + 19%主目标最大生命；均物理（蛋免控·只吃原地伤） |

**核对**：打击 / 龟派气波 / 强化普攻被动 / 龟盾(击飞带击退·已融被动) 已对照用户 #475 后实时原话 → 一致；过肩摔/不屈数值 = 代码值。**未发现 Claude 自造。**

## 2. 糖果 (candy) — A级 · 近战
**基础**：HP 966 · ATK 40 · DEF 15 · MR 16 · 射程 70 · 移速 105 · 攻速间隔 0.85s
　（⚠ 原代码是"远程射程400"= Claude 自造，用户原话"挥锤砸向目标"近战 → 已改近战 70，见 46b4b06）

**被动（双被动锁死）**
- **甜蜜吸取（战斗内）**：登场时对最肥敌（最大生命最高）吸取其 25% 最大生命 → 全回复自己（不杀死，最低留 1）
- **糖果罐（局外赛季）**：选糖果龟当统领才有；大轮 1 颗；打碎按当前计数领档奖（赢+1/输+4·封顶30·6档收菜）

**普攻 · 糖果拳**：1.1×ATK + 5%自身最大生命 物理 + 目标攻击-15%(5秒)（近战）
　（⚠ 原代码普攻带"15%吸血自愈"= Claude 自造，用户原话普攻**无自愈** → 已删；自我回血在糖果锤+甜蜜吸取）

**3选1**
| 技能 | 龟能 | 完整效果 |
|---|---|---|
| **糖果锤** | 80 | 蓄力举锤 → 猛砸一条直线 200码内的敌人，伤害由命中敌**分摊**：总 1.8×ATK + 12%自身最大生命 物理（均分）；回复造成伤害的 40% 生命 |
| **糖衣炮弹** | 120 | 船长 R 式：在场上单位最密集区降炮弹雨，半径 600码，持续 4s（每 0.5s 一跳=8跳）；每跳友军获 2%糖果龟最大生命护盾、敌军受 0.2×ATK+2%糖果龟最大生命 魔法 + 减速20%(每次受击刷0.5s) |
| **糖果炸弹** | 召唤 | 战斗开始召唤实体炸弹(HP=50%糖果龟最大生命·每秒衰减8%·不攻不动)；每次本技龟能满→炸弹活着则喂续命(+25%上限+治疗10%)、死了则重召(20%)；炸弹死亡爆炸=150%炸弹最大生命均摊魔伤 |

**核对**：甜蜜吸取/糖果锤/糖衣炮弹/糖果炸弹 对照用户原话一致；普攻远程+自愈 = Claude 自造已改回近战无自愈（46b4b06）。
