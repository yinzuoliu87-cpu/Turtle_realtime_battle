# 实时版 战斗基础 — 标准知识库（单一事实源）

> 本文是实时版战斗的**全局规则标准**，所有数值/公式以此为准。规则改了就在这里替代旧的，别让旧记录散在别处造成分歧。
> 逐龟策划评审流程：① 基础数值（确认无缺失）→ ② 被动（实装/效果/视觉/bug）→ ③ 每个技能（范围/龟能花费/完整效果/视觉/bug）。
> 逐龟的具体数值/技能 → `逐龟数据表.md`（数据化）；技能视觉 → `技能视觉规格.md`。

## 二、伤害 / 暴击公式（焊死）
**有效护甲/魔抗（穿透：先 % 后 flat，可为负）**
```
有效护甲 = def × (1 - 护甲穿透%) - 护甲穿透flat
有效魔抗 = mr  × (1 - 魔抗穿透%) - 魔抗穿透flat
真实伤害：无视护甲/魔抗（有效值 = 0）
```
**减伤倍率（常数 K = 40）**
```
有效防御 ≥ 0 → 倍率 = 1 - 防御/(防御 + 40)          (减伤)
有效防御 < 0 → 倍率 = 1 + |防御|/(|防御| + 40)      (增伤, 上限~2×)
```
**最终伤害**
```
最终 = 基础 × 减伤倍率 × (1 + 攻击者增伤%) × (1 - 受害者减伤%)   ※真伤不吃减伤%
```
**暴击**
```
暴击率有效封顶 100%；触发暴击则伤害 × 暴伤倍率
暴伤倍率 = 1.5 + max(0, 暴击率 - 1.0) × 1.5      (溢出100%每1%→1.5%暴伤)
全龟通用。
```

## 三、等级系统
```
主属性(生命/攻击/护甲/魔抗) × (1 + 0.05×(等级-1))
攻速                         × (1 + 0.02×(等级-1))
不吃等级：暴击/暴伤/穿透/闪避/吸血/治疗加成/护盾加成/减伤/增伤/反伤/韧性
```
- **召唤体**用主人的等级；**装备效果**按等级缩放；技能里的 `{等级}` 公式用等级算。
- **来源**：现阶段 = **图鉴右上角调试器**（设全体 1/2/5/10）；正式版从外部网站；对手用快照等级。
- 技能解锁：idx3 需 Lv4、idx4 需 Lv7。

## 四、普攻原则
- **所有龟普攻 = 单段（hits=1）**，一下下打。倍率逐龟定（小龟 = 1.0×ATK 物理）。
- **每段叠层的连带**（线条墨迹/水晶结晶/闪电电击/龟壳溅射）：普攻单段后逐龟评审定（保持多段特例 / 单段叠多层 / 单段叠1层）。

## 八、其它全局常量
- **龟 hp = 最终值**（pets.json 直接写；2026-06-29 全龟血量 **×0.7 降30%** 平衡，已烤进数据）
- **装备 hp = 最终值**（装备STATS 已 ×3 烤进数据，不再代码 ×3）
- HP_MULT = 3（仅作 base↔final 比率：召唤体 raw 值 ×、装备"%自身最大生命"回收 maxHp/ 用它；不再作用于龟/装备 hp）
- 暴伤基准 = 1.5（全龟）
- 减伤常数 K = 40
- 护盾上限 = maxHp × SHIELD_CAP_MULT（保留现值）
- CC 基础时长：stun/freeze 1.5s、buff/嘲讽等按现值，吃韧性缩减

## 九、飘字（数字）规矩 — 1:1 回合制 `_spawn_float_text`/`VisualConstants`
- **颜色按伤害类型**：物红 `#ff4444` / 魔蓝 `#4dabf7` / 真白 `#ffffff` / 治绿 `#06d6a0` / 盾白。走 `VisualConstants`，**技能不准各传主题色**（`_resolve_dmg` 记类型，`_apply_damage_from` 飘字处取色）
- **动画**：伤害 = 爆大 pop(1.6/1.8/2.2/2.5 按量级) → hold → **抛物弹射**(朝屏边 jump_x、jump_y 上抛、**重力 200** 先上后下) → fade；治疗/护盾 = pop 1.2 + 缓升 50px(sine) 1.5s
- **暴击**：类型色不变、只字号变大；内嵌 crit 图标必 `expand_mode = EXPAND_IGNORE_SIZE`（20×20，否则原图撑爆）
- **同时跳的数字按类型排行错开** `_float_row_offset`：红 rank0 / 蓝 rank1 / 白 rank2，220ms 窗口紧凑 ×**15px**（常量 `FLOAT_ROW_GAP`；实时版定值，比回合制 22px 更贴近，用户 2026-06-29 确认；缺色不留空）

## 十、技能视觉规矩
- **技能名不准当文字飘**（业余）—— 已全局删 63 个；只留 亡灵/复活/赛季淘汰 事件
- **施法不用飘空 billboard 图标**
- **逐技能定制特效 = 港回合制现成的**（竹叶生命球 `_spawn_bamboo_orb`、治疗绿光 `_play_heal_glow`…）→ 2.5D 化（billboard/3D 弧/贴地环）
- 受击闪光默认白，可 per-unit 改色（`_flash(u, col)`；竹叶充能那击 = 敌绿闪）
- 触发时机对上视觉：如竹叶生长 回血/成长 延到**绿球落到身上**才生效

## 十一、数据驱动（每龟读自己数据，不跑通用硬编码）
- **护盾/治疗技**：`pets.json skillPool[].fx`（shieldAtk/shieldHp/healHp/buffDef/buffMr/buffDur）+ 技能名，`_cur_skill_data` 读
- **龟能花费**：读 `skillPool[].energyCost`（per-skill），缺则 `SkillEnergy` 类型兜底

## 十二、朝向
- 立绘**默认朝左**；`flip_h=true` 朝右
- **跟移动方向**：移动中朝移动方向；不动保持上次；初始 左队朝右 / 右队朝左（朝敌）

## 十三、龟能（技能资源）= 冷却
- 龟能固定速率充能；技有龟能花费；攒够才放；**"冷却秒数" = 花费 × 0.075** 换算（非独立计时器）。全 UI 显"龟能N"不准出现"CD"
- **自/友向技**（护盾/治疗/增益/变身，见 `_SELF_CAST_SKILLS`）：**任意距离即放**；敌向技 + 普攻：要进射程
- 单一事实源 `SkillEnergy`（类型）+ per-skill `energyCost`（数据驱动覆盖）
