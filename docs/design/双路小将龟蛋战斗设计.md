# 双路 + 小将 + 龟蛋 战斗设计（定稿 2026-07-05，用户逐点确认）

> 把双路/小将/龟蛋真正接进 3D 战斗（`RealtimeBattle3DScene`）。数值全**沿用回合制**（`Turtle-Project-L_Demo/games/turtle-battle-godot` 只读参照，绝不写它）。旧口述版已被本定稿覆盖。

## 一、整体流程（用户定）
匹配到异步对局 → **上半场战场**（放上路 3 只·场内自由摆位 → 开打）→ **下半场战场**（下路 3 只 → 摆位 → 开打）→ **1-1 才进终极战场**（上下路**幸存·含小将·带 30% 回血·无人数上限** → 摆位 → 开打）。
- **6 单位 auto_split 成上路 3 + 下路 3**（两拨不同的单位，**不是同队打两半场**；上半场幸存**不进**下半场）。
- 2-0 横扫 → 下半场后直接结束；1-1 → 终极。
- egg 血**两路累积**，**谁蛋先破谁输**。

## 二、局外配置（背包改造）
- **去掉"阵容格子"概念**。改成 **上战场 / 下战场 + 6 头像（3 统领 + 3 小将）**，玩家把 6 个分到上/下战场（各 3）。
- **小将两种类型**（点头像切换）：**前排小将 = 近战挥砍 ×1.4** / **后排小将 = 远程射击 ×1.5**。⚠"前排/后排"**只是小将作战类型标签，不是站位**（站位靠场内自由放置）。
- **统领没有前后排**，按自己龟性（近战/远程 = STATS.melee/atk_range）行动。
- **自动精英**：某路 **0 统领**（3 个都是小将）→ 该路**首个小将变精英**（`is_elite`，整排均摊 `eliteRowSplit`）。
- 小将局外**只能配前后排类型**，不换装备/技能（只有普攻）。

## 三、场内放置
- 每进一个战场，先在**自己半场的放置区**把该路单位**自由拖到位置**（非格子吸附）摆好 → 点"开打"。
- 三个战场（上/下/终极）都各放置一次。终极放上下路全部幸存（无上限）。
- 对手（bot/快照）自动布阵。

## 四、龟蛋 + 围栏（用户新设计，回合制无围栏）
- 蛋 = **真 fighter**：`hp/maxHp = 2000 + 100×平均等级`（跨路累积，带上路打剩的）；atk/def/mr=0 纯血包；`_isEgg`+`_eggImmune`（免控/免斩/免嘲讽）；走完整伤害管线。
- **全息围栏**：平时罩住蛋 → **不可被主动索敌**（敌不自动锁它），但 **AoE / 增益能穿围栏碰到蛋**（平时就能靠 AoE 累积削蛋 = 策略空间）。
- **团灭 → 围栏消失 10 秒** → 蛋可被自由攻击 10 秒。
- **结束条件**：非终极路——10 秒窗口结束就本路结束，**蛋破也只累计不立刻输**（剩血带下路/终极）；终极路——**蛋破 = 直接输**（无时限），败方蛋 **×5 承伤 + 每 2.5s（≈1回合，EQ_TICK）自损 25%maxHP**。
- 蛋在**左右两端基地**+围栏；单位在中间打；放置区在自己半场；AoE 要够到基地才碰得到蛋。

## 五、小将数值（沿用回合制 phase2_minion）
HP `250×mult×3=750`、ATK `30×mult`、双抗 7；`mult=1.05^(等级-1)`；前排挥砍 ×1.4 / 后排射击 ×1.5；精英整排均摊；只有普攻无技能被动；`_isMinion` 免永久 buff/跨路继承。

## 六、对手
混用 **bot（随机阵容）+ ghost 快照**，自动布阵。先 bot 跑通再接快照。

## 七、分阶段实施
- **P1** 局外背包改造：去阵容格子 → 上/下战场 + 6 头像分配 + 小将前后排 toggle + 自动精英；存 `GameState`（不动战斗）
- **P2** 场内放置阶段：进战场先放置区自由摆 → 开打（三战场）
- **P3** 解锁 `REVIEW_DEMO` + 分路 spawn：读分配 spawn 当前路 leaders+小将（前排近战/后排射击/精英）+ 对手 bot/快照；单路跑通
- **P4** 蛋=真 fighter + 围栏（不可索敌 + AoE 穿栏）+ 团灭 10s 窗口；一路"团灭→打蛋→窗口"跑通
- **P5** 半场/终极流程：top→bottom→final；幸存+30% 回血进终极；蛋 HP 跨路累积；终极 ×5+自损；谁蛋先破谁输
- **P6** HUD（蛋血条/路指示/等级）+ 平衡 + F5
