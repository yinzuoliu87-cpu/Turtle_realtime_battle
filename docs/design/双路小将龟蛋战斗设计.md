## 一、整体流程（用户定）
匹配到异步对局 → **上半场战场**（放上路 3 只·场内自由摆位 → 开打）→ **下半场战场**（下路 3 只 → 摆位 → 开打）→ **1-1 才进终极战场**（上下路**幸存·含小将·带 30% 回血·无人数上限** → 摆位 → 开打）。
- **6 单位 auto_split 成上路 3 + 下路 3**（两拨不同的单位，**不是同队打两半场**；上半场幸存**不进**下半场）。
- 2-0 横扫 → 下半场后直接结束；1-1 → 终极。
- egg 血**两路累积**，**谁蛋先破谁输**。

## 二、局外配置（背包改造）
- **去掉"阵容格子"概念**。改成 **上战场 / 下战场 + 6 头像（3 统领 + 3 小将）**，玩家把 6 个分到上/下战场（各 3）。
- **小将两种类型**（点头像切换）：**前排小将 = 近战挥砍 ×1.4** / **后排小将 = 远程射击 ×1.5**。⚠"前排/后排"**只是小将作战类型标签，不是站位**（站位靠场内自由放置）。
- **统领没有前后排**，按自己龟性（近战/远程 = STATS.melee/atk_range）行动。
- **自动精英**：某路 **0 统领**（3 个都是小将）→ 该路**首个小将变精英**（`is_elite`，整排均摊 `eliteRowSplit`）。
- 小将局外**只能配前后排类型**，不换装备/技能（只有普攻）。

## 三、场内放置
- 每进一个战场，先在**自己半场的放置区**把该路单位**自由拖到位置**（非格子吸附）摆好 → 点"开打"。
- 三个战场（上/下/终极）都各放置一次。终极放上下路全部幸存（无上限）。
- 对手（bot/快照）自动布阵。

## 四、龟蛋 + 围栏（用户新设计，回合制无围栏）
- 蛋 = **真 fighter**：`hp/maxHp = 2000 + 100×平均等级`（跨路累积，带上路打剩的）；atk/def/mr=0 纯血包；`_isEgg`+`_eggImmune`（免控/免斩/免嘲讽）；走完整伤害管线。
- **全息围栏**：平时罩住蛋 → **不可被主动索敌**（敌不自动锁它），但 **AoE / 增益能穿围栏碰到蛋**（平时就能靠 AoE 累积削蛋 = 策略空间）。
- **团灭 → 围栏消失 10 秒** → 蛋可被自由攻击 10 秒。
- **结束条件**：非终极路——10 秒窗口结束就本路结束，**蛋破也只累计不立刻输**（剩血带下路/终极）；终极路——**蛋破 = 直接输**（无时限），败方蛋 **×5 承伤 + 每 2.5s（≈1回合，EQ_TICK）自损 25%maxHP**。
- 蛋在**左右两端基地**+围栏；单位在中间打；放置区在自己半场；AoE 要够到基地才碰得到蛋。

## 五、小将数值（沿用回合制 phase2_minion）
HP `250×mult×3=750`、ATK `30×mult`、双抗 7；`mult=1.05^(等级-1)`；前排挥砍 ×1.4 / 后排射击 ×1.5；精英整排均摊；只有普攻无技能被动；`_isMinion` 免永久 buff/跨路继承。

## 六、对手

## 六·四、经济 / 装备 / 等级 / 入口出口（用户 2026-07-05 定）
- **经济 = 只大轮/赛季经济**：局外商店买卖、3合1（`auto_merge_all` 扫 bench+persistent_equipped 合并池，3 同 id 同星→升 1 星，MAX 3）、双路布阵；**局内纯战斗，无半场间商店/攒币/升级**。
- **装备**：**leader 和小将都能配装备**（局外配，进战斗生效）。小将装备是**实时全新**（回合制硬编码不能给小将配装, id 共享）：每个小将在 `dual_lineup` 存自己的 `equips`，**装备槽数跟 leader 一样按等级给**（`equip_slots_for_level`）；进战斗走同一装备注入管线。
- **缩放规则**（对齐回合制, agent 已扒 file:line）：
  - **蛋**：HP = `2000 + 100 × 赛季等级`（**只有蛋这样线性吃等级**）
  - **leader**：`×(1 + 0.05×(等级-1))` 缩 HP/攻/防/抗（沿用回合制 ×5%/级）+ 装备叠加
  - **小将**：`750/45/7 × 1.05^(等级-1)`，前排挥砍×1.4 / 后排射击×1.5 攻 + 装备（新）
  - 等级源 = **大轮赛季等级**（固定, 无局内升级; B 版无 dual_level 买经验）
- **对局入口/出口**：主菜单「开始对局」→ 匹配(`setup_dual_lane`+ghost) → 进双路战斗(`dual_active` 替代 `DUALLANE` env)；战斗结束 → 赛季结算(命/币/胜场) → 回主菜单 → 局外商店/布阵 → 下一局。
- **蛋死亡破碎动画**：PixelLab 新做（裂→碎逐帧，接蛋死亡钩子）。回合制蛋动画是 idle/挨打的。

## 六·五、地图 + 障碍物（用户 2026-07-05 追加）
- **技术做法**：海床地面=大 `PlaneMesh`+PixelLab 海床瓦片纹理平铺；障碍/墙/基地=2.5D billboard 立绘(PixelLab 生 礁石/沉船/穹顶)+底部导航障碍 footprint；navmesh=`NavigationRegion3D`(地面挖障碍 footprint)，单位移动改沿 `NavigationServer3D.map_get_path` 路点走(2D pos↔3D 用 `_world_pos`)，保留分离/绕友；相机角度不动(2.5D 斜视)场地放大就往后拉；**3 版主题**(深海礁盘/沉船遗迹/火山海沟)=换海床纹理+障碍立绘+布局，F5 挑一。
- **地图 = 深海对称竞技场**：两端基地（蛋 + 半透明发光穹顶围栏）+ 中间一大障碍（沉船/大礁）+ 两侧墙，形成掩体/通道。场地放大，相机**固定俯视框住全场**。
- **障碍物 = 墙 + 大障碍**，**只挡移动**（弹道/技能穿过，不挡）。
- **避让 = A. navmesh 导航寻路**（用户定）：把 Godot NavigationServer3D 接进现有自定义移动——烘焙 `NavigationRegion3D`（地面挖掉障碍），每单位 `NavigationServer3D.map_get_path` 沿 navmesh 绕障碍到目标，`_do_move` 跟路点走（保留分离/绕友）。
- **美术**：程序化几何 + PixelLab 生贴图（海床/礁石/沉船/蛋/围栏）。
- **放置区**：你半场一块区域自由拖拽摆位，自动避开障碍/基地。
