## 轮 1 · 字体

**核过的现状**（不是猜的）
- `assets/fonts/m6x11.ttf`（英文/数字像素字）、`assets/fonts/NotoSansSC-Regular.otf`（中文，**已打包**）
- `assets/themes/default_theme.tres`：`FontVariation_main` = m6x11 打底 → fallback `[NotoSansSC, SystemFont(雅黑/苹方/…)]`
  → **CJK 早已解决**（含 Web：Noto 是打包的，不依赖系统）
- ❌ **无任何 emoji 字体**。UI 大量使用 🍬💠🗡🔼🐢 等。Windows 上靠 `allow_system_fallback` 命中 Segoe UI Emoji；
  **Web / Linux / macOS 导出会掉字形（豆腐块）**。
- ❌ `MainMenuScene._bold_font()`：中文粗体只挂 `SystemFont`，**没有打包的 Noto 作为回退** → 无雅黑的机器上粗体中文可能掉字形。

**要做**
1. 引入 **Noto Emoji**（SIL OFL，可随游戏分发）；优先单色版（体积小、Godot 稳）。
2. `default_theme.tres` 的 fallback 链尾加 emoji 字体。
3. `_bold_font()` 回退链补上打包的 Noto SC（在 SystemFont 之前）。
4. **自证**：写 `tests/verify_fonts.gd` — 关掉系统回退，逐字符 `has_char()` 检查 CJK 常用字 + 项目里实际用到的 emoji 全集（从源码里扫出来）。

### 轮1 结果（✅ 完成 · commit 见下）
- **CJK 其实早就做好了**（`NotoSansSC` 已打包 + SystemFont 回退），路线图那条"CJK+emoji 字体捆绑"里**只有 emoji 是真缺的**。
- 新增 `assets/fonts/NotoEmoji-Regular.ttf`（1.98MB，SIL OFL，附 `NotoEmoji-OFL.txt`）。
- `default_theme.tres` 回退链：`m6x11 → NotoSansSC → SystemFont(CJK+系统回退) → NotoEmoji`。
  **NotoEmoji 排最后**是有意的：桌面端先吃系统彩色 emoji（Win=Segoe UI Emoji / mac=Apple Color Emoji），
  只有 Web/Linux 取不到系统字体时才落到打包的单色版 → **既不豆腐块、桌面又是彩色**。
- 修 `MainMenuScene._bold_font()`：原来中文粗体只挂 `SystemFont`，**没有打包字体兜底** → 无雅黑的机器上粗体中文会掉字形。现补 `NotoSansSC + NotoEmoji`。
- **测试抓到 3 个真缺口**（不是猜的）：`↺`(U+21BA) `⛶`(U+26F6) `✕`(U+2715) —— Noto SC / Noto Emoji / NotoSansSymbols2 都不全覆盖。
  为 2 个字符再拖一个字体不划算 → 换成回退链已覆盖的：`↺→🔄`(U+1F504)、`✕→✖`(U+2716)；
  `⛶` 无等义替代 → **去掉图标只留文字**（"全屏 / 退出全屏"）。
- **扫描器口径**：只取 **字符串字面量** 里的字符（注释不渲染，不计入）。
  〔踩坑：我第一版扫全文，结果我自己写的"解释替换"的注释里又含 `↺⛶✕`，测试假红了一轮。〕
