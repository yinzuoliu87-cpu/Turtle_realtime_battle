# 实时版 视觉方向方案 v1

> 回答两件事: ① 2D 合不合适 ② 出一版视觉方向。立项 2026-06-27。

## 一、2D 到底合不合适？→ 合适，且是正确选择（不是将就）

**商业先例（全是 2D 俯视、全是千万级吸金大成功）：**
- 元气骑士 / 元气骑士前传(Soul Knight)、吸血鬼幸存者(Vampire Survivors)、Archero(弓箭传说)、Brotato、战魂铭人、土豆兄弟…
- → **2D 俯视动作/自走棋是成熟的商业品类**，不是低端选项。

**为什么 2D 对我们是对的：**
1. **深度不依赖 3D**：我们的核心是自走棋战术（走位/CC/连招/阵容/养成），2D 俯视完全表达得了（吸血鬼幸存者就是纯 2D 走位）。
2. **3D 成本小团队扛不住**：建模/绑骨/骨骼动画/海量美术。Botworld 是 Featherweight（Rovio 系大厂）的 3D，比产能拼不过、也没必要拼。
3. **已有 2D 立绘**：转 3D = 全扔重做。
4. **2D juice 一样有肉感**：拉伸压缩/闪白/顿帧/粒子/震屏/软影 在 2D 全套成立，元气骑士的打击感就靠这些。
5. **移动端友好**（目标上 App Store）：2D 包小、省电、全平台跑得动。

**核心心法**：**做最棒的 2D 俯视自走棋（自成商业级），借 Botworld 的「设计」(机制/深度/CC/连招/养成)，不抄它的「3D 渲染」。不需要 3D。**

## 二、出一版：推荐方向 = 2D「3/4 高角度斜俯视」+ 卡通深海

**视角：3/4 斜俯视**（元气骑士/Archero/Botworld 都是这个角度）
- 不是 90° 正俯视（那个看龟壳顶、做不出"飞起来"），也不是纯平。
- 立绘**直立朝镜头**，地面有透视。→ 既看清**走位**（自走棋命脉），又能**伪 3D 做高度/击飞/juice**（软影+抬升，立绘离开影子=飞起来）。
- Botworld 本身也是 3/4 角度；我们用 2D 模拟这个角度 = 同样观感、零 3D 成本。

**美术调性：圆润鲜艳卡通**（跟现有龟立绘走，同 Botworld 那种友好/彩色/有光泽），非像素、非写实。
**主题：深海/水底**（斗龟场）—— 竞技场是水底场地（珊瑚/沉船/光柱/气泡），不是纯色矩形。

## 三、各层把"占位"换成"商业级"
| 层 | 现在(占位) | 商业级目标 |
|---|---|---|
| 竞技场 | 纯色矩形 | 深海主题场地(透视地面+边界+环境+光影气泡) |
| 单位 | 裸立绘 | 立绘billboard + 软影(随高缩放) + idle呼吸浮动 + 受击闪白/压扁 + 朝向 |
| UI | 裸ColorRect血条 | 描边血条/能量条/技能图标/风格化飘字 |
| VFX | 几乎没有 | 卡通特效层(斩击弧/弹道拖尾/冲击火花/CC图标/击飞尘) |
| 打击感 | 位置跳一下 | 全套juice: 预备/拉伸压缩/缓动/软影/闪白/顿帧/粒子/震屏/音效 |

## 四、落地阶段（先搭"商业级地基"再铺内容）
1. 相机定 3/4 视角 + **软影系统**（伪3D地基，所有高度/击飞靠它）
2. 深海竞技场场地美术
3. 单位表现层（billboard+影+idle+受击反馈）
4. UI 重做（血条/能量/飘字/技能图标）
5. VFX + juice 系统（**击飞当标杆做全套**，再推广全局）
6. 音效
