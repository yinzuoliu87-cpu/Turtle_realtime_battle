# 装备演示 / 测试配置登记表（EQDEMO）

> 每件需要专属演示的装备，在这里登记它的**完整测试配置** + **一键运行命令**。
> 以后要看某件装备效果 = 照抄命令、设好环境变量、实际跑起来看，不用每次重新搭场景。

## 机制：EQDEMO 测试台

战斗场景 `RealtimeBattle3DScene` 内置一套 env 门控的装备演示台（`REVIEW_DEMO` 评审模式的一个分支，正常游戏不受影响）。设 `EQDEMO_EQUIP=<装备id>` 即启用：

| 环境变量 | 作用 | 默认 |
|---|---|---|
| `EQDEMO_EQUIP` | 要演示的装备 id（如 `p2eq_034`）；**必填**，设了才进演示台 | — |
| `EQDEMO_STAR` | 装备星级 1/2/3 | `2` |
| `EQDEMO_CARRIER` | 携带者龟 id | `basic`（友方假人） |
| `EQDEMO_DUMMYHP` | 右侧敌方假人血量 | `5000` |
| `EQDEMO_FAST` | =1 时大熊约5秒快速出现放波（专给看特效，仅 p2eq_034） | 关 |
| `EQDEMO_WAVEROT` | 冲击波波纹朝向偏移（度），调弹道方向用 | `0`（顺行进方向） |
| `EQDEMO_ATTACKER` | =1 携带者正常移动+普攻+技能（测 on-hit/on-cast 类件；否则是不动的持装备假人）| 关 |
| `EQDEMO_SKILL` | 强制携带者只放指定技能 key（如 `rainbowStorm`），测 AoE 技×装备交互 | 无（AI自选）|
| `EQDEMO_COUNT` | 给携带者塞 N 件同款（测多件叠加）| `1` |
| `EQDEMO_ENEMY1` | 敌1距携带者的码数 | `210` |
| `EQDEMO_GAP` | 两敌间距（码）；溅射/AoE 类件把假人摆近 | `500` |

**默认场景布局**（写在 `_spawn_teams` 里）：
- **左 = 1 只友方假人携带者**：`no_move` + `no_basic` + 移速0 + 50万血，只负责持装备触发效果（不会自己乱跑/攻击，方便观察召唤/触发）。
- **右 = 2 只敌方假人**：`no_move` / 无技能 / 30 双抗 / 会掉血；**相距 500 码**（`_cx+60 + i*500`），方便看直线/位移/穿透类效果。
- 装备注入：`_inject_equipment` 对 `REVIEW_DEMO` 早返回，但 **EQDEMO 例外**（否则携带者拿不到装备）。

## 运行命令

Godot 可执行文件：`C:\Users\Louis\Desktop\Godot_v4.6.3-stable_win64.exe`
仓库：`c:\Users\Louis\Documents\GitHub\turtle-realtime-godot`

**开窗口看（推荐，墙钟卡时长，能看动作/位移）**——Git Bash：
```bash
timeout 42 env EQDEMO_EQUIP=p2eq_034 EQDEMO_STAR=2 \
  "/c/Users/Louis/Desktop/Godot_v4.6.3-stable_win64.exe" \
  --path "c:/Users/Louis/Documents/GitHub/turtle-realtime-godot" \
  res://scenes/RealtimeBattle3D.tscn --resolution 1280x720
```
> ⚠ 别用 `--quit-after N`（按帧算，窗口没锁 60fps 会提前关）；用 `timeout <秒>` 卡墙钟时长。
> ⚠ headless（`--headless`）只用来抓 parse / 运行时报错，**sim 时间不可靠**，看不到真实节奏。

## 已登记装备

### p2eq_034 玩偶小熊（5费传说） — 大熊召唤 + 小菊式冲击波
- **配置**：`EQDEMO_EQUIP=p2eq_034 EQDEMO_STAR=2`（默认即此配置：左友方假人携带者移速0，右2敌假人相距500码）。
- **预期时间线**（2星，需3层）：≈4/8/12s 各出一只小熊踢击飞 → 满层后携带者**金光蓄力1.2s** → 召**大熊** → 大熊走到假人**熊掌**（攻速0.5=每2秒一下）攒2层 → **冲击波**（600码、波速500px/s 慢）直线穿过相距500的两假人：1.5ATK物理 + 击飞0.8s + 滞空期朝熊平滑拉回（击飞态不能移动/充能）。
- **看点**：小熊节奏、蓄力停顿、大熊攻速、冲击波速度/射程/穿透、拉回是否平滑不重叠。

### p2eq_001 锈蚀短剑（1费普通） — 就绪型武器(射程=携带者射程)
- **配置**：`EQDEMO_EQUIP=p2eq_001 EQDEMO_STAR=2 EQDEMO_CARRIER=hunter`（远程携带者射程400；敌1@210码在射程内、敌2@710码在射程外）。近战演示改 `EQDEMO_CARRIER=basic EQDEMO_ENEMY1=60`（射程70内）。
- **预期**：每3秒就绪，射程(=携带者射程)内有敌→斜砍(0.6/0.75/1.0×ATK+40/60/100×暴击率)物理；面向镜头斜砍斩弧。
- **看点**：3秒节奏、只打射程内的敌(远的不打)、斜砍弧角度。

### p2eq_003 锋利鲨齿（3费稀有） — 命中溅射(150码)+圈圈扩散
- **配置**：`EQDEMO_EQUIP=p2eq_003 EQDEMO_STAR=2 EQDEMO_ATTACKER=1 EQDEMO_GAP=70`（攻击型携带者普攻→溅射旁边70码的假人）。
- **AoE 技交互演示**：`EQDEMO_CARRIER=rainbow EQDEMO_SKILL=rainbowStorm EQDEMO_ATTACKER=1 EQDEMO_GAP=70`（彩虹龟全色风暴每 tick 命中→触发溅射+扩散环）。
- **预期**：每次命中→对目标 150 码内相邻敌溅射 15/28/50%×本段伤害物理 + 金色扩散环；多件叠加。
- **看点**：溅射到相邻假人、圈圈扩散、AoE 技每段都触发。

### p2eq_004 暴君之牙（4费史诗） — 处决 + -999999真伤大字
- **配置**：`EQDEMO_EQUIP=p2eq_004 EQDEMO_STAR=2 EQDEMO_CARRIER=hunter EQDEMO_ATTACKER=1 EQDEMO_DUMMYHP=500`（低血假人才够快打进斩杀线）。
- **预期**：hunter 打假人到血 <斩杀线(5/7/10%maxHP+10/15/40%×暴击率) → 处决秒杀 + 跳固定 **-999999 真伤大字**（实际伤害=剩余血）+ 回 20 龟能飘字。
- **看点**：处决触发时机、-999999 大字、回龟能。
