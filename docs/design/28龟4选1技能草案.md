## ✅ 决议（用户 2026-06-27）
1. **4选1 候选可含被动**：候选 = 该龟的【主动技 + 被动】混合。**选了被动 = 强化那个被动**（非"放出来"的技）。**选的若无主动部分 → 该龟不攒龟能、无主动施放**（纯被动+普攻）。→ 故下方"新拟·普攻升格"作废，缺主动技的龟改用它的**真实被动**填候选位（待按各龟 passive 重列候选）。
2. **3 改造定**：财神「招财进宝」→ **全体范围回血（不耗币）**；糖果「糖果罐」→ **全体范围回血**（先占位）；缩头「指挥」→ **随从立即追加一次施法**（认）。
3. **变身触发（逐龟定）**：双头 = **手动**（变身=它的一个主动技候选，选了/放了才变）· 熔岩 = **攒怒气**（满→变火山）· 赛博 = **死亡时**变机甲 · 龟壳 = **按时间**（如 10s 觉醒）。
4. **仍待定**：财神除招财进宝外的币技（骰子产币 / 梭哈耗币 / 打击两下按币数）—— 实时战斗里**到底有没有"币"这套经济**？没有的话这几个也得改。
