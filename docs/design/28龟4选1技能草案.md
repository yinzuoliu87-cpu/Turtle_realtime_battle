# 斗龟场实时版 — 28龟「4选1」候选（权威版·被动替代普攻升格）

> 依据 = `data/pets.json`（逐龟 `passive` 字段 + `skillPool` 内 `passiveSkill`/`enhancesPassive` 标记）。

## 规则（用户 2026-06-27 决议）
- **普攻 = 自动「打击」**（与 4选1 分开；候选里剔除基础"打击/攻击"）。
- **4选1 候选 = 真·主动技 + 被动 混合**。选**主动** → 龟能条涨满放一次→再涨再放(循环)；选 **[被动] = 强化该被动**，且**该龟不攒龟能/无主动施放**（纯被动+普攻）。
- 被动两类（列候选都算 [被动]）：**基础被动**(`passive` 字段) + **强化被动**(`enhancesPassive`，本就是被动强化版)。
- **战中有币**：每秒 +2 + 单位死亡给币（沿用回合制）→ 财神币技保留。
- 计时全改秒（起步值，**F5 调**）。**3 改造**：财神招财进宝/糖果罐 → 全体范围回血；缩头指挥 → 随从立即追加施法。**变身**：双头手动/熔岩攒怒/赛博死亡/龟壳按时间(~10s)。

## 关键
1. **不再缺候选**：剔除普攻后每龟都满 4（多为 3 主动 + 2 被动），**无需新拟**。部分龟列了"/可选被动"备选。
2. **变身龟的变身被动建议占第4候选位**：熔岩「熔岩之心」/赛博「浮游炮」/龟壳「气场觉醒」；双头变身是主动技(切换近战)。
3. **回合制专属剔除**：水晶「不朽」(存活第10回合) 已剔。其余 DoT/控制/buff「回合」→「秒」。
4. **3 改造已落实**：财神招财进宝/糖果罐→全体回血、缩头指挥→随从追加施法。

## 回合制 → 实时 转换对照（技能实装时逐技过）
| 回合制机制 | 实时改法 |
|---|---|
| 时长"N回合"(冰寒/眩晕/buff/DoT/护盾) | "N秒"(重调值) |
| "本回合行动X次"(缩头指挥) | 剔除/改造(→随从立即追加施法) |
| "回合末结算"(DoT) | 每秒 tick |
| "存活到第N回合"(水晶不朽) | 直接剔除 |
| "每N回合触发"(龟壳气场第4/8回合) | "每N秒/开战N秒" |
| "全员行动后"(水晶球) | 独立单位定时射 |
| 回合制开奖/抽取(财神/糖果) | 改实时(范围回血/召唤) |
| 攒怒气/星能/层数 | 实时累积(造伤/受伤/时间) |
