## 提炼的节奏铁律
- **攻速 0.5–1.1 秒**：追击最快0.5(脆皮高DPS)，坦克/狙击/辅助0.8–0.9(多数)，闪避最慢1.08(远程风筝)。**攻速不随"肉度"变慢——坦克照样0.8** ；慢的是远程kiter。
- **大招 CD 4–12 秒**(中位~9)，**副技 CD 4–8 秒** → 一局每 bot 放技能数次。
- **移速比** 最快/最慢 = 20(闪避)/9(坦克) ≈ **2.2×** ；快=闪避/追击，慢=坦克。
- **射程比** 远/近 = 10(狙击)/~2(近战) ≈ **5×** ；闪避居中3.5-6。
- **TTK**：脆皮追击 HP7880÷DPS2392 ≈ **3.3s 秒人**；坦克 HP2.3万耐久；差异主要靠 **DPS(6.5×幅度) + HP(3.3×幅度)**，不是攻速。3v3 集火一局 **~20–30 秒**。
