## §1 赛季 / 大轮
- **一个大轮 = 一个赛季**：8 命起（`hearts`），5 天到期（`_P2.SEASON_DURATION_SEC`）。到期 `ensure_season()` → `start_new_season()` 滚下一轮。出处 `autoload/GameState.gd: ensure_season / is_season_expired / start_new_season`。
- **新大轮重置**：`hearts=8` / `season_total_battles=0` / `season_level=1` / `season_xp=0` / `meta_deepsea_coins=0` / `season_leaders=[]` / `loadouts={}` / 背包糖果罐等全清。养龟站 `pet_levels` 不动。
- **大轮等级**：每场 +2 XP（`add_season_xp`），升级到 `MAX_LEVEL=10`。商店出货档/装备槽都读 `season_level`（`equip_slots_for_level=clamp(ceil(lv/2),1,5)`）。
- **命 / 淘汰**：输一场 `lose_heart()`（clamp≥0）；0 命 `is_eliminated()` → 战斗变**表演赛**（无消耗继续打，`RealtimeBattle3DScene: _last_was_exhibition`）。⚠**0 命目前不自动开新大轮**（待拍板）。

## §2 大轮阵容锁定 + 3选1 技能可变（R1, 2026-07-11）
- **锁定源** = `GameState.season_leaders` 满 `REQUIRED_PETS(3)` 只有效龟。持久化在 `save()`。
- **选龟界面自检两态**（`TeamSelectScene`）：
  - `season_leaders` 空（新赛季/首次）→ **全选模式**：自由选 3 龟 + 每龟 3选1 技能。`_on_start` 写入 `season_leaders` 并锁定。
  - `season_leaders` 满 3 → **确认出战模式**（`_roster_locked=true`）：阵容预填灰显、候选池半透明、所有改阵容入口（点卡/点槽/拖拽/清空/上次）守卫拦截（只看详情），仅 3选1 技能可切；确认按钮「⚔ 确认出战」。
- **3选1 技能**：`GameState.loadouts[pet_id]=idx`（1=默认签名/2/3=候选），持久化（JSON 数字回来 float→强转 int）。战斗引擎 `_resolve_chosen_index` 读它选技；28 龟 idx1-3 全部 `impl:true` 且都在战斗门 `_IMPL_SKILLS` 内 → 选谁真出谁，不静默回落。**拆掉了旧的 `_toggle_skill` 里 `idx!=1` 死码**（曾一律挡候选技）。
- **重选时机**：仅赛季滚动（5 天到期）。大轮内阵容锁死、技能每场可调。出处 `TeamSelectScene: _season_roster_ready / _roster_locked / _toggle_skill`，`GameState: loadouts/season_leaders`。

## §3 匹配（异步 ghost + bot，按档位）
- **按档位**：`Backend.bracket_for_battles(season_total_battles)` → 9 档（0-8）：0-1场=档0 / 2-3=档1 / 4-5=档2 / 6-8=档3 / 9-14=档4 / 15-20=档5 / 21-30=档6 / 31-40=档7 / 40+=档8。
- **抽对手**：`find_opponent(bracket)` = 同档 ghost 池抽（`pool_find`，排除自己）→ 池空/全排除则 `make_bot(bracket)` 兜底（随机 3 龟 + 按档装备/等级）。永远有对手。出处 `scripts/engine/backend.gd`，`MatchmakingScene`。
- **ghost 池**：`user://ghost_pool.json`，按档分桶（每桶封顶 50）。玩家每场后 `upload_ghost` 上传自己阵容快照（`RealtimeBattle3DScene:11088`）。
- **★内置 10 支策划种子队**（`data/ghost_seed.json`）：按档 0-7 分桶，冷启动/老档 `load_pool` **幂等并入**（`seed_` 标记防重复），下次 `upload_ghost` 落盘。名单：新手渔夫/浅滩过客(档0) · 珊瑚卫队/潮汐商会(档1) · 暗礁赌徒(档2) · 机械神教(档3) · 星界游侠(档4) · 龟壳军团(档5) · 熔火宗师(档6) · 海渊霸主(档7)。**按档上装备**：档0=每龟 1 件普通★1 → 档7=每龟 3 件（史诗×2+传说 ★3），件数/稀有度/星级随档升。
- **对手接线**（关键，2026-07-11 修）：实战是双路模式，对手走 `_dual_foe_lane(lane)` —— 从 `dual_ghost.lane_assign[lane]` 取该路 leaders + 挂各自 `dual_ghost.equipped[pet]` → 走既有 `_dl_equips` → `_inject_equipment` 应用装备。**此前它查 `dual_ghost["top"]`（快照没这键）一直回落写死 8 龟池，匹配抽到谁/装备全没用上**——已修。
  - ⚠ **对手等级仍 = 玩家 `season_level`**（`_unit_level` 右队 base=season_level），非 ghost 自身；**对手技能走默认签名技**（右队 `_chosen_skill_types(id, use_loadout=false)`，不读 ghost loadouts）。两者待拍板是否按档/按快照。

## §4 战斗内 UX（R2, 2026-07-11）
- **结束按钮化**：结算横幅两个 Button「⚔ 再战」(`reload_current_scene`) /「🏠 返回菜单」(`change_scene MainMenu`)。键盘 R/ESC 保留作桌面快捷不显文案。出处 `RealtimeBattle3DScene: _show_banner / _make_result_btn`。
- **暂停**：顶右 ⏸ → `get_tree().paused` + 暂停浮层（继续/重开/返回菜单）。按钮/面板 `process_mode=ALWAYS`（暂停中可点）；重开/返回前先解 paused（否则新场景冻）；结算后禁暂停。出处 `_build_pause_log_ui / _toggle_pause / _build_pause_panel`。
- **战斗日志**：顶右 📜 → 可滚动富文本。`_log` bbcode 封顶 `_LOG_CAP=200`（防无限增长）。低频钩子：`_cast_skill`(放技) / `_kill`(主龟阵亡，召唤体/蛋不刷屏) / `_show_banner`(胜负)——**不逐次出伤刷屏**。

## §5 战斗统计（照回合制样式，R2c, 2026-07-11）
- **累计口径**（每单位 dict 字段，实际值）：`_st_dealt`(输出，攻方实际造成) / `_st_taken`(承受) / `_st_heal`(实际回血 hp-hb) / `_st_shield`(实际获盾) / `_st_crit`(暴击次数) / `_st_kills`(击杀主龟数)。按伤害类型分桶 `_st_dealt_by_type`/`_st_taken_by_type`：`_dmg_bucket(raw,col)` = raw→真实(tru) / 蓝#4dabf7→法术(mag) / 否则物理(phy) / 无来源→DoT。
- **战中「分段条」面板**（📊 切）：暗底金棕边浮层，4 Tab（造成/承受/治疗/护盾，选中白/未选灰），我方绿#06d6a0 / 敌方红#ff6b6b 双列，各按当前 Tab 值**降序**；每行 名+值 + stacked 分段条（高 12 圆角 4，空轨 rgba(1,1,1,.05)）。造成/承受按类型堆叠：物理红#ff4444 ‖ 法术蓝#4dabf7 ‖ 真实+DoT 白；治疗绿#06d6a0 / 护盾青#58d3ff（alpha 同回合制）。0.4s 自刷。
- **结算 7 列表**：龟 / 出伤 / 受伤 / 治疗 / 暴击 / 击杀 / 剩余(hp/maxHp)；金表头#ffd93d，名前稀有度色点，存活白#ffffff / 阵亡灰#888888+「(阵亡)」。
- **召唤体单列一行**（用户拍板）：浮游炮/随从/大熊在代码里确实是 `is_summon`，统计里各占独立一行（名前「↳」缩进），归本队，不并入主人。出处 `_build_dmg_stats_panel / _render_dmg_stats / _make_ds_bar / _ds_parts / _build_stats_panel / _stats_column`。

## §6 局外四界面
- **商店** `ShopScene`：售价 = 装备 cost 1:1 深海币；买后 `auto_merge_all` 自动三合一。
- **背包** `InventoryScene`：点装备选中→点龟装/卸；装/卸/买后**自动三合一升星**（`GameState.auto_merge_all`，无"一键合星"按钮——已删死码/误导文案）；龟身槽按 `equip_slots_for_level`。
- **图鉴** `CodexScene`：龟/装备/学派/状态/规则 五 Tab。龟能是单一事实源；**已删陈旧「小商店物品池」页**（回合制局内金币商店，实时无）；**规则 Tab 保留**（`battle_rules` 待改制后加入战斗）。
- **战绩** `RecordScene`：记 `record_match(result, lineup, "实时", int(_t))` → 显「实时」+「N 秒」（`MODE_LABEL` 回退命中，本就正确）。
- **返回**：四界面 + 图鉴均响应 ESC（`_unhandled_input` ui_cancel → MainMenu，R3 统一）。
