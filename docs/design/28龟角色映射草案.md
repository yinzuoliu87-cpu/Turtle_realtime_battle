## ⚠ 待用户审的几点
1. **🔴 闪避(风筝) 0 只 = 最大空缺**。没有龟基础值是"快+脆+远程风筝"。最佳候选 **幽灵龟**（虚化自带闪避+魔法远程）→ 建议从狙击挪去闪避，填满 7 角色。（次选猎人/忍者）
2. **🟠 狙击/辅助 各 6 偏多、追击仅 2 偏少**。骰子/赌神（现格斗）也很"冲脸脆皮"，可挪 1 只去追击。
3. **近战~12 / 远程~16**：远程略多（法术系天然远程），可接受。
4. **换形龟（双头/熔岩/赛博/龟壳）**：变身怎么触发（攒能量？时间？）→ 角色可能随形态变，待定（阶段4）。
5. **召唤系（海盗/糖果/赛博/水晶/缩头）**：召唤物实时下变独立单位走位，会改实战定位 → 优先原型验手感。

# #7 攻速 / 移速 / 射程 默认值草案（起步值，F5 调）

> 28 龟基础值压得很窄 → 三个**新属性**按 近战/远程 + 属性倾向(肉/中庸/脆) 分档。**全是起步值，实时手感靠 playtest 调。**

## 分档
**移速**：
- 慢 70：石头·钻石·泡泡·水晶·缩头（坦克5）
- 中 105：小龟·竹叶·天使·寒冰·财神·彩虹·海盗·糖果·凤凰·赛博·宝箱·龟壳（12）
- 快 145：忍者·双头·幽灵·骰子·赌神·猎人·线条·闪电·熔岩·星际·无头（脆11）

**攻速**：
- 快 0.65：忍者·闪电·线条·猎人·幽灵·骰子（连击/连发6）
- 中 1.0：小龟·竹叶·天使·寒冰·财神·彩虹·海盗·糖果·凤凰·赛博·双头·赌神·熔岩·星际·无头（15）
- 慢 1.5：石头·缩头·钻石·宝箱·水晶·泡泡·龟壳（重击/坦克7）

**射程**：
- 近战 70：小龟·石头·竹叶·忍者·钻石·财神·宝箱·缩头·无头·龟壳·双头·熔岩·彩虹·水晶（14）
- 远程-中 230：寒冰·海盗·糖果·泡泡·凤凰·赛博·骰子·天使·赌神（9）
- 远程-远 340：幽灵·猎人·线条·闪电·星际（单体狙击5）

> **原则**：脆皮跑得快、坦克最慢；连击系普攻快、重击/坦克普攻慢；单体狙击射程最远、AoE/辅助远程稍近、近战必须贴脸。数值全待原型 F5 调手感。
