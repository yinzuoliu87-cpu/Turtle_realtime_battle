## ⚠ 待用户审的几点
1. **🔴 闪避(风筝) 0 只 = 最大空缺**。没有龟基础值是"快+脆+远程风筝"。最佳候选 **幽灵龟**（虚化自带闪避+魔法远程）→ 建议从狙击挪去闪避，填满 7 角色。（次选猎人/忍者）
2. **🟠 狙击/辅助 各 6 偏多、追击仅 2 偏少**。骰子/赌神（现格斗）也很"冲脸脆皮"，可挪 1 只去追击。
3. **近战~12 / 远程~16**：远程略多（法术系天然远程），可接受。
4. **换形龟（双头/熔岩/赛博/龟壳）**：变身怎么触发（攒能量？时间？）→ 角色可能随形态变，待定（阶段4）。
5. **召唤系（海盗/糖果/赛博/水晶/缩头）**：召唤物实时下变独立单位走位，会改实战定位 → 优先原型验手感。
