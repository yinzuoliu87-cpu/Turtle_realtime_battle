## 怎么审（F5 流程）
1. 我把 `RealtimeBattle3DScene.gd` 顶部 `REVIEW_TURTLE` 设成受审龟、`REVIEW_SKILL_IDX` 设成受审技的 skillPool 索引（**0=普攻 / 1=技1 / 2=技2 / 3=技3 / -1=默认轮转**）。
2. 你在编辑器 **F5**（debug 构建 → REVIEW_DEMO 自动开）→ 进「开始战斗」→ 场上是**受审龟 vs 打不死的假人**，反复放这一个技，你盯着看。
3. 你说「过 / 要改（哪里不对）」→ 我改（或记 F5 再看）→ 切下一个技。**一只全过再下一只。**

> ★**每技记「演示配置」**（用户 2026-07-11）：有些技需要特定假人布局才能看全（如龟派气波要"1贴脸+1远处"才逼出智能冲刺）。配置写在代码 `REVIEW_DEMO_CFG["受审龟:skill_idx"]`（相对受审龟的假人 dx/dy），并在本账本每技记一份。缺省=REVIEW_DUMMY_COUNT 横排。
> ⚠ 被动多是属性/光效类，有些没有独立"演示"，看战斗中触发时的表现。
> ⚠ 每改一处特效我不说"已好"，只说"改了，F5 看"（视觉必你眼验）。
