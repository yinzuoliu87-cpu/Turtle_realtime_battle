2. 你在编辑器 **F5**（debug 构建 → REVIEW_DEMO 自动开）→ 进「开始战斗」→ 场上是**受审龟 vs 打不死的假人**，反复放这一个技，你盯着看。
3. 你说「过 / 要改（哪里不对）」→ 我改（或记 F5 再看）→ 切下一个技。**一只全过再下一只。**
