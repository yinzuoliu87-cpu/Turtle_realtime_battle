# 深海龟战 · 双路 — 全流程 + 核心逻辑总览

> 当前实装状态的"地图"，供理清结构 / 讨论设计方向用。2026-06-26 整理。

## 一、整体流程（一局怎么走）
```
主菜单
  → 匹配 (搜对手 2.2s + VS 展示 2.2s, 可取消)
  → 选龟界面 (选 3 只龟 · 每龟 5 选 3 技能 · 看羁绊预览 · 确认阵容)
  → 大地图 (双路龟蛋攻防 · 暗分路: 把 3 龟分到 上路/下路)
  → 战斗: 上路一局 → 下路一局 → 终极战场(final)
  → 结算 (胜负 / 退出回主菜单)
```
跨场景带的数据：阵容(left_team/slots) · 敌队(right_team, 随机生成) · 等级/币(局内) · 装备(equipped_p2, 跨路保留) · 蛋血。

## 二、核心结构：双路 (dual-lane)
- 一场比赛 = **上路(top) + 下路(bottom)** 两条战线，各打一局，再终极战场决战。
- 目标：**凿穿对方龟蛋**（蛋 = 攻防核心单位，免疫处决/控制）。
- 跨路持久：等级/币/装备 跨路继承；**蛋血跨路回 30%**；装备战中改动回写保留。每局内独立升级。
- **终极战场(final)**：两路打完的决战（蛋×5 + 自损 25%，200 回合硬上限防死循环）。
- **决胜局**：30 回合不分胜负 → 每回合全龟 +1 层怒气(每层 +30% 伤害) + 疲惫(治疗/护盾 ×0.5)，逼出胜负。

## 三、单位
- **龟（统领）**：玩家选的 3 只。各有 技能(5选3, 花龟能) · 被动 · 属性 · 装备槽 · 类型/学派(羁绊归属)。
- **小将**：每路开打自动补位到 3 名（前排砍 1.4× / 后排射 1.5×）；某路 0 统领 → 一只升「精英」(整排均摊伤害)。不跨路继承。
- **龟蛋**：攻防核心，跨路带血，免疫处决/控制。
- **召唤物**：部分技能/装备召唤（熊/触手/亡魂/虫/幽灵龟…）。

## 四、战斗逻辑（回合制 · 玩家操控）
- **side-based 整队制**：一方所有可动单位轮流出手 → 换边。
- **玩家出手**：选龟(点脚下发光龟) → 选技能(够龟能才放) → 选目标 → 执行。30s 倒计时，超时 AI 代打。点 ⓘ 看详情。
- **AI**：敌方由 AI 控制（约 1.7s 思考，后续手缩到 0.7s）。
- **出手后链路**：on-hit 链(吸血/反伤/受击盾/破甲/控制…) → side-end(DoT/召唤/装备结算/学派回合盾治疗) → round-begin(羁绊/类型钩子) → 换边/下回合。
- **资源 = 龟能(energy)**：攒着放技能（**非冷却制**）；普攻不花龟能。

## 五、build 系统（云顶式）
- **商店**（战中常驻）：5 卡货架，按等级费用概率掷货(1-5 费)，**刷新恒 2 费**，买装入备战席。
- **备战席**（横排在战场下方）：装备库。拖到龟身 = 装备；拖到商店 = 卖(亏 ~20%)；**3 张同 id+星 = 合 1 升星**(1→2→3 星)。
- **经济**：每回合 +10 币 + 持有 20% 利息(cap10)；局内等级(经验)花币升；AI 也买装(席不超上限)。
- **等级**：局内升级 → 解锁单位槽 / 更高费率概率。

## 六、羁绊（双维度！认知负担最大的一块）
- 每件装备**同时**属于 **1 个类型(12 职业)** + **1 个学派(11 种族)**：
  - 类型：剑 · 盾 · 枪 · 法器 · 药水 · 护符 · 弓箭 · 食物 · 奇械 · 饰品 · 灵物 · 遗物
  - 学派：唤灵 · 潮汐 · 深渊 · 圣甲 · 深海军械 · 珊瑚 · 血牙 · 远古 · 黑礁 · 极地 · 玄甲
- 场上同类型/学派装备越多 → 激活越高档（各有 2/4/6 或 3/6/9 档）。
- 效果 = 属性加成(剑+攻/盾+护甲/护符+魔抗…) + 机制(唤灵召唤亡魂/潮汐回血/深渊腐蚀/圣甲圣盾/军械炮台/远古觉醒…)。

## 七、内容量
28 龟 · 140 技能 · 59 装备 · **12 类型 + 11 学派 = 23 条羁绊** · 状态(眩晕/冻结/灼烧/流血/腐蚀/僵硬/嘲讽/恐惧…)。

---

## 设计层张力（讨论用，非实装问题）
1. **想同时是两种游戏**：云顶式 build/经济(自动战斗为前提) + 手操回合制战斗 —— 两者乐趣互相稀释，节奏拖、学习成本翻倍。
2. **系统过载**：双维度羁绊(23 条) + 技能 + 龟能 + 经济 + 决胜局 + 蛋 + 小将。云顶只 1 个羁绊维度。
3. **PvE 打随机 AI**：缺自走棋的 8 人 PvP 社交/竞争张力，stakes 与重复可玩性薄。
4. **双路意义**：若只是"打完上路打下路"= 重复 2 倍，需跨路取舍才值。

**核心待定问题**：这游戏最该让玩家爽的那一下是什么？(凑爆装 combo / 手操秒人 / 别的) —— 定了它，系统取舍(砍什么、做透什么)才清楚。建议先把"快+清晰"版做完玩 3-5 局、记"爽点 vs 拖点"，再定方向。
