> 这是阶段4 实装的**权威参照**：每只龟的技能经用户逐个评审、回合制机制已转实时后的最终定义。

## 通则（用户 2026-06-27 确认）
- **护盾 = 永久吸收**（打光为止，不设时长）。
- **"N回合"时长**（buff/控制/DoT）→ **"N秒"**。
- **站位语义**（横排/同列/前排/后排）→ **直线/范围内/最近/拉近**（自由移动下）。
- **累积值**（岩层/星能/墨迹/层数）→ **实时累积**（造伤/受伤/时间）。
- 普攻 = 各龟 0 能量基础技（自动平A，不入 4选1）。选主动→龟能涨满放循环；选被动→不涨龟能。
